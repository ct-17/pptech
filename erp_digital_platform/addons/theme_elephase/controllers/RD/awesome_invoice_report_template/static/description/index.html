Awesome Invoice Report Template

Simple Beautiful and Easy

This module allows you to generate professional invoices with different colors. Very simple and easy to set up.
With or without Shipping address, It will generates automatically an invoice depends on your information in the back-end.
You can make a combination of 2 colors in order to get an awesome invoice.

How to install it

In order to install this module, you need to follow this steps

  • Copy dependent module inside the folder "Modules" and put it inside the addons folder of your odoo
  • Delete "Modules"
  • Put also the folder awesome_invoice_report_template inside the addons folder.
